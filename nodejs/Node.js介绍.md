# 一. 概述
- Node.js采用事件驱动和非阻塞I/O模型

# install 

```
Mac安装Node.js
brew install node

Ubuntu 上安装 Node.js
$ sudo git clone https://github.com/nodejs/node.git
sudo chmod -R 755 node
cd node
sudo ./configure
sudo make
sudo make install
node --version

Ubuntu apt-get命令安装命令格式如下：
sudo apt-get install nodejs
sudo apt-get install npm

```
